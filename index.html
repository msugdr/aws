AWS

AWSは面白いよ! Git との連携をしていなかったので一回してみようと思った。 そこで remote にリポジトリを作ろうとしてハマってしまった。 以前にやったときは、GitHub の Web 上で新しいリポジトリを作って、 そこへコマンドプロンプトからアクセスした。 今回は、コマンドプロンプトからリモートのリポジトリを作成しようとしたら うまくいかなかった。かなり基本的なことだと思うのだが。

いろいろ調べると、実はそういう風にはなっていないのではなか。 自分が触れるマシン環境にしか作れない。リモートリポジトリというのは、 遠隔との共有（繋がりがあること）を指しているだけのようだ。 だからリムーブしても繋がりが除去されるだけで。どのコンテンツも消えない。 そうか！そういうことなんだ。
